**Day 3**
## Automatic Gate System with Keypad

**Date:** 01.04.2026

**Description of Today's Activities:**

Today, we worked on designing and implementing an **automatic gate system** using a **keypad interface**. We focused on both the **hardware and simulation aspects** of the project:

1. **Keypad-controlled Gate:**  
   - Programmed a keypad to control the opening and closing of the gate.  
   - Implemented password verification to allow authorized access only.  

2. **Simulation in Tinkercad:**  
   - Designed the gate system virtually using **Tinkercad**.  
   - Tested how the servo motor responds to correct and incorrect passwords.  
   - Verified the logic and movement of the gate before actual implementation.  

This activity helped us understand the integration of **microcontrollers, input devices, and actuators**, as well as the importance of **simulation before physical deployment**.
