## Day 5: Smart Room Automation & Real-Time Environmental Monitoring System

### 📝 Description  
On Day 5 of the internship, we designed and conceptualized an advanced smart room automation system integrated with real-time environmental monitoring using the ESP32 microcontroller and the Blynk IoT platform.

The system was developed to simulate intelligent control of multiple electrical appliances while simultaneously tracking key environmental parameters. Four LEDs were utilized to represent controlled devices within the room: the first and second LEDs simulated two independent fans, while the third and fourth LEDs represented two separate lighting units. This multi-device setup demonstrated how modern smart systems can manage multiple loads efficiently and independently.

In addition to automation, the system incorporated environmental sensing capabilities. The Blynk IoT dashboard was configured to display real-time data, including temperature, humidity, and soil moisture levels. These metrics are essential for maintaining optimal environmental conditions and are widely applied in smart homes, server rooms, and agricultural monitoring systems.

The integration of control and monitoring into a single IoT platform provided practical insight into how embedded systems communicate with cloud-based applications. This project emphasized scalability, remote accessibility, and the importance of user-friendly interfaces in modern automation systems.

By the end of the session, we strengthened our understanding of IoT architecture, real-time data visualization, and multi-device control, all of which are fundamental in building efficient and intelligent smart environments.

### 🛠️ Components Used  
- ESP32 Microcontroller  
- DHT Sensor (Temperature & Humidity)  
- Soil Moisture Sensor  
- LEDs (4) – representing fans and lamps  
- Resistors  
- Jumper Wires  
- WiFi Connectivity  
- Blynk IoT Platform  

### ⚙️ System Functionality  
- Controls multiple devices independently (fans and lamps)  
- Displays real-time temperature readings  
- Displays real-time humidity levels  
- Monitors soil moisture conditions  
- Provides remote control via mobile application  
- Enables real-time data visualization on IoT dashboard  
- Demonstrates integration of automation and environmental monitoring
