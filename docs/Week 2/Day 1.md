# Week 2 - Day 1: Introduction to PHP & Web Dashboard

## Description
On the first day of Week 2, we were introduced to **PHP**, a server-side scripting language for web development. We focused on creating a **web dashboard** for our DHT sensor project to monitor temperature and humidity.

### Objectives
- Learn basics of PHP for web development
- Set up a local development environment (XAMPP)
- Retrieve sensor data from Arduino using PHP
- Build a web dashboard interface
- Implement simple admin login functionality
- Understand data flow from IoT sensors to the web

### Key Activities
1. Installed and configured XAMPP for local PHP development
2. Created PHP scripts to fetch DHT sensor data
3. Designed a dashboard interface using HTML, CSS, and PHP
4. Tested real-time display of temperature and humidity
5. Implemented admin login to secure dashboard access

### Outcome
By the end of the day, we had a **working web dashboard** displaying live sensor data, ready for monitoring temperature and humidity remotely.
